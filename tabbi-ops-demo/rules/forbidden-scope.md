# Forbidden Scope

Do not implement login/auth, roles, real backend, real AI provider, API keys in mobile app, fiscal billing, Mercado Pago integration, printer integration, WebSockets, offline-first, push notifications, drag & drop floor editor, visual floor editor, inventory management, Ant Design, SCSS Modules, DOM APIs, Pointer Events, CSS Grid, Woki desktop interactions, EAS build or Play Store publication.

If a request implies one of these items, stop and suggest a smaller MVP-safe alternative.
