# UI Style Rules

- Use orange as primary color.
- Do not use violet as brand color.
- Use light background.
- Use white cards.
- Use large rounded buttons.
- Use chips for filters.
- Use table status colors.
- Keep prices and totals visible.
- Use short Spanish operational copy.
- Prioritize readability over decoration.
- Do not create dark mode by default.
- Do not make the app look like fintech/SaaS.
- Do not copy desktop layout into mobile.
